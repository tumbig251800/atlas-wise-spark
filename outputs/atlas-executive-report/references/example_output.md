## คำถามที่พบบ่อย

**Q: ถ้าไม่มีข้อมูลพอสำหรับ KPI บางตัว ทำอย่างไร?**

A: ระบุใน KPI card ว่า "ไม่มีข้อมูล" หรือ "N/A" และอธิบายใน Method Notes tab
ห้ามใส่ตัวเลขจาก reference มาแทน

**Q: ถ้า layout ในไฟล์ตัวอย่างต่างจาก report_layout.md ให้ยึดอะไร?**

A: ยึด `references/report_layout.md` เป็นหลัก เพราะเป็น spec ปัจจุบัน
ไฟล์ตัวอย่างใช้ดูเป็น reference เสริมเท่านั้น

**Q: ถ้าผู้ใช้อยากได้ Google Sheets แทน Excel ทำอย่างไร?**

A: สร้างไฟล์ Excel ก่อน แล้วถามผู้ใช้ว่าต้องการ:
  1. ดาวน์โหลดเป็น .xlsx ไว้ใช้เอง
  2. อัปโหลดเป็น Google Sheets ใหม่
  3. อัปเดต Google Sheet ไฟล์เดิม (ต้องให้ ID หรือ URL)
ห้ามอัปโหลดโดยไม่ถาม
